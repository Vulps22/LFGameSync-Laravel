Privacy Policy

Last Updated: June 18, 2025

This Privacy Policy describes how LFGameSync ("the bot", "we", "us", or "our") collects, uses, and shares information when you add our bot to your Discord server and use its commands. By using LFGameSync, you agree to the collection and use of information in accordance with this policy.

1. Information We Collect

To provide its core functionality, LFGameSync collects and stores the following types of information:

Information You Provide Directly

  • Your Game Library: When you use the /library add command or click the "I have this game" button, we store a link between your Discord User ID and the specific game(s) you add.
  • Submitted Game URLs: When you use the /register game command, we store the store page URL you provide to add a new game to our global database.

Information Collected Automatically

  • Discord User ID: We use your unique Discord ID as the primary identifier to manage your personal game library and settings.
  • Discord Server ID: We store the ID of the servers you and the bot are both members of. This allows the bot to know which users are in which server and to manage your per-server sharing preferences.
  • Sharing Preferences: We store your choice for the /sharing on or /sharing off command for each server individually. This preference is stored as a link between your User ID and the Server ID.

Information from Third Parties

  • Public Game Store Data: When a game is registered via a URL, the bot "scrapes" publicly available information from that store page, such as the game's name and image. We currently support scraping from Steam, GOG, and the Meta Quest store.

What We DO NOT Collect

We do not collect or store your Discord messages, email address, password, or any personal information beyond what is listed above.

2. How We Use Your Information

We use the information we collect for the following purposes:

  • To provide our core functionality, such as executing the /lfg command to find other players in your server.
  • To allow you to build and manage your game library.
  • To operate and improve the service, including troubleshooting bugs and preventing abuse.
  • To share anonymous, aggregate statistics (specifically, our total server count) with third-party bot listing websites like Top.gg.

3. How We Share Your Information

Your information is shared in the following ways:

  • With Other Users in Your Server: If you use the /sharing on command, other members of that same server will be able to see that you own a specific game when they use the /lfg command. If sharing is off, your library is hidden in that server.
  • With Third-Party Services: We automatically report our total server count to Top.gg for ranking purposes. This is an aggregate, anonymous number.
  • For Legal Reasons: We may disclose your information if required to do so by law or in response to valid requests by public authorities.

We will not sell or rent your personal information to any third party.

4. Your Rights and Controls

You have full control over your data within LFGameSync:

  • Control Sharing: You can control the visibility of your game library on a per-server basis using the /sharing on and /sharing off commands at any time.
  • Remove a Game: You can remove any game from your personal library using the /library remove command.
  • Request Data Deletion: You can request the complete deletion of all your stored data (your User ID and all associated library entries) by contacting the Data Controller.

5. Data Security

We are committed to protecting your data. We use a secure database to store the collected information, and access to this database is strictly limited. All application secrets and credentials are kept confidential and are not stored in the public repository.

6. Data Controller and Contact Information

For the purpose of data protection regulations, the Data Controller for LFGameSync is the Discord user vulps, who can be contacted through direct message on Discord or by opening a support ticket on the official LFGameSync server.
